# Code of Conduct

## Our Pledge
We are committed to making participation in this project a respectful and harassment-free experience for everyone.

## Expected Behavior
- Be respectful to all contributors
- Accept feedback professionally
- Communicate clearly and politely
- Support teamwork and collaboration

## Unacceptable Behavior
- Harassment or insulting comments
- Personal attacks
- Discrimination of any kind
- Publishing others' private information

## Enforcement
Project maintainers are responsible for clarifying and enforcing this code of conduct.
