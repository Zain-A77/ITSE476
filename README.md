Welcome to  **OpenShelf** repository!  

Our project aims to build a simple and organized web-based library management system that helps manage books, users, and borrowing activities in a more efficient digital way. At the same time, this repository is designed to reflect the principles of **open-source software development**, including collaboration, transparency, documentation, version control, and issue tracking.

## 📖 About This Project

We are **Software Engineering students** working on this project as part of our **Free and Open-Source Software Development** course.  
The main purpose of this repository is to apply the concepts of **open-source collaboration** and **version control** using Git and GitHub in a practical course project.

This repository represents our teamwork and learning journey throughout the semester, where we focus on:

- Open-source project organization
- Collaboration between team members
- Version control using Git and GitHub
- Documentation and project transparency
- Tracking updates, changes, and issues

## 🌐 Website Overview

**OpenShelf** website is a simple **Online Library Management System** created for academic purposes.  
It provides a basic idea of how a digital library platform can be organized to manage books and related activities in a clear and structured way.

## 🎯 Project Focus

In this course, our main focus is not only on creating the project itself, but also on understanding how software projects are managed in an **open-source environment**.  
Through this repository, we aim to demonstrate proper use of:

- GitHub repositories
- Branching and version control
- Issue tracking
- Team collaboration
- Open and clear documentation
